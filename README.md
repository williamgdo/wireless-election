# Eleição Wireless

Esse projeto é uma implementação de eleição em redes sem fio, usando python.  

Para executar:  
```
python setup.py
```

Após iniciar os terminais, apertar Enter em algum deles para iniciar uma eleição.  

É necessário ter o programa xterm instalado.
